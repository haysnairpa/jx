# J-Explorer

A Simple Mobile app for exploring Jababeka area using React Native & Expo.

## Features
- Location Search
- Route Navigation
- Search History
- Firebase Integration

## Setup
1. Clone repo
2. Copy .env.example to .env and fill in the credentials
3. Install dependencies: `npm install`
4. Run: `npm start`

## Tech Stack
- React Native
- Expo
- Firebase
- React Navigation
